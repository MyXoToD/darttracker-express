# Darttracker

## Tech Stack

- Frontend: Angular
- Backend / API: Express
- Database: MySQL
- Deployment: Docker

## Tooling

- Linter
- Prettier
- Dev Server
- SQL Schema Auto setup
- One line run command for whole app
- Docker Compose

## Setup

- docker compose up --build -d
